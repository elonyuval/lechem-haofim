# עדכון תוכן האתר

כל תוכן האתר (תפריטים, מחירים, פרטי עסק) נמצא בקבצים בתיקייה הזו. אין צורך לגעת בקוד של הקומפוננטות כדי לעדכן תוכן.

## איך לעדכן מוצר בתפריט (`products.ts`)

1. מוצאים את המוצר לפי `id` (או מוסיפים אובייקט חדש לפי אותה הצורה).
2. עורכים `nameHe`, `description`, `price`.
3. מסמנים `isPlaceholderPrice: false` ו-`isPlaceholderText: false` כדי שהתג "תוכן לדוגמה" ייעלם.
4. להוספת תמונה אמיתית: מוסיפים שדה `imageUrl` עם נתיב לתמונה תחת `public/images/products/`, ומסמנים `isPlaceholderImage: false`.

## איך לעדכן קטגוריה (`categories.ts`)

עורכים `nameHe`/`description` ישירות — אלה כבר תואמים לאתר הקיים.

## איך לעדכן פרטי עסק (`businessInfo.ts`)

טלפונים, כתובת, אימייל, קישורי סושיאל וכשרות — הכל בקובץ אחד. שינוי כאן משפיע אוטומטית על ה-Header, ה-Footer, דף "צור קשר", והודעות ההזמנה שנשלחות בוואטסאפ/מייל.

## איך לעדכן חבילות אירוח (`hostingPackages.ts`)

אותו עיקרון: עורכים את השדות, ומסמנים `isPlaceholder: false` כשהתוכן אמיתי.

## הזמנות שהתקבלו

הזמנות לא נשמרות בקבצים האלה — הן נשמרות אצל הלקוח בדפדפן (localStorage) דרך `src/lib/orderStorage.ts`. ראו את `Order` בקובץ `types.ts` למבנה המדויק.
